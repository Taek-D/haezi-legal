# haezi-legal-site

GitHub Pages로 서비스 이용약관/개인정보 처리방침을 호스팅하기 위한 정적 페이지입니다.

## 배포(가장 쉬운 방식)
1. 이 폴더를 새 GitHub repo에 업로드
2. GitHub → Settings → Pages
3. Build and deployment:
   - Source: Deploy from a branch
   - Branch: main / folder: /docs
4. 저장 후, 아래 URL이 생성됩니다:
   - https://<username>.github.io/<repo>/
   - 약관: https://<username>.github.io/<repo>/terms.html
   - 개인정보: https://<username>.github.io/<repo>/privacy.html

## 토스 콘솔에 입력할 URL 예시
- 서비스 이용약관 URL: https://<username>.github.io/<repo>/terms.html
- 개인정보 처리방침 URL: https://<username>.github.io/<repo>/privacy.html
